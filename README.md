## Описание

Проект написан на Django web-фреймворк Python.


### Требования
- python3.8+
- Django==3.1.7

### Установка

1. Скачать и установить python (для Windows)
    - [Официальный сайт www.python.org](https://www.python.org/downloads/)
    - (Не забудь поставить галочку при установке типа "ADD to PATH")
    
2. Разархировать проект на рабочий стол.
3. Открыть командную строку (Win+R, набрать cmd, после Enter)
4. Перейти в проект
```bash
cd C:\Users\{you}\Desktop>cd project
```
Подсказка: по команде dir выведутся все файлы проекта в командную строку.

5. Установить зависимости через командную строку.
```bash
pip install -r requirements.txt
```
6. Провести миграцию для создания таблиц в базе данных (выполнять команду в директории с файлом manage.py)
```bash
python manage.py migrate
```
7. Создать суперпользователя
```bash
python manage.py createsuperuser
```
8. Запустить сервер
```bash
python manage.py runserver
```

- Главная страница по адресу 127.0.0.1:8000
- Страница администратора(для создания проектов) 127.0.0.1:8000/admin/ (Потребует логин/пароль из шага 7)
- Отображаются проекты только с задачами.

- Если забыли пароль, то можно удалить файл db.sqlite3 и создать суперпользователя заново.
